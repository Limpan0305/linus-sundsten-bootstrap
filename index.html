Begreppsförklaring

Element

Ett element är en komponent av din html som kan bestå av taggar, t.ex. h1, p, a, div osv. De flesta element skrivs med en starttagg och avslutas med en stängningstagg.


HTML

HTML är en förkortning på Hyper Text Markup Language. HTML är den tet en sida består av. HTML är den text som bygger upp en sida och som gör att sidan kan ha olika funktioner.


ID

ID är ett attribut man sätter på ett element för att specificera det så att man kan använda sig av just det elementet i t.ex. din CSS. ID kan endast användas en gång per html dokument.


Class

Class är ett attribut likt "ID" som benämning på ett element men skillnaden är att Class kan användas flera gånger inom samma html dokument. Du kan använda class på olika element för att benämna de olika.


Meta

Meta är en del som endast syns i HTMlen och inte på din sida. Uppgiften för meta är att beskriva hur dokumentet ska läsas.


Responsive

Responsive är att sidan komprimeras desto mindre den blir. Om man gör sidan mindre ska ändå strukturen på sidan bevaras. Ett exempel är att göra en sida mobilanpassad.


Http

Http är det kommunikationssättet som webbsidor använder sig av för att överföra sina sidor över WWW. Http sätter sig runt informationen som ska skickas och för fram den via World Wide Web.


Https

Https är liknande http men skillnaden är bokstaven "s" som läggs till. Bokstaven "s" står för security och gör är att den sätter säkerhet runt datan vilket göt det svårare att läsa datan som skickas över WWW. Det som utgör säkerheten är att den krypteras vilket gör det svårare att läsa av infon.


Compressed data

Compressed data är engelska för datakompression. Datakompression är en förminskning av datan men man behåller samma data. Du kompirmerar filen så att den tar mindre plats men med samma innehåll. Detta gör det lättare att transportera filer och att det går snabbre.


CDN

CDN kallas även för innehållsleveransätverk. Det är ett nätverk med utspridda närverk av proxy-servrar som ökar driftsäkerhet och prestanda men också miskad latens. I dagens samhälle används CDN till all typ av webb, video, grafik, streaming osv.


Local bootstrap

Local bootstrap är ett lokalt bibliotek med flera tusen rader av css. När man länkar till local bootstrap så används bootstrapbiblioteket till din sida. Det som är speciellt med bootstrap är att din sida blir responsive. Man kan skriva över local bootstrap med din egen css genom att länka den nedanför.


JQuery

JQuery är ett javascript-bibliotek med kod som kan användas för att förenkla din HTML och CSS. Jquery används idag på 85% av dagens webbsidor.


Flexbox

Flexbox är ett sett att ordna upp ditt innehåll i kolumner och rader. Du använder flexbox för att på ett enkelt sätt hantera boxars storlek och position. Med flexbox kan du göra sidan responsive.


Grid

Grid är så som en webbsida är indelad. Med header längts upp och footer längst ner. Grid kan vara hur du placerar saker i kolumner och på rad. Grid layout kan se väldigt olika ut på olika webbsidor men oftast följer de standarden.


Code snippets

Code snippets är en del av återanvändbar kod. Denna kod kommer antingen från källkod, maskinkod eller text. De används oftast i stora programmeringsmoduler.
