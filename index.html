Begreppsförklaring

Element

Element är en komponent av din html som kan bestå av taggar, t.ex. h1, p, a, div osv.

HTML

HTML är en förkortning på Hyper Text Markup Language. HTML är den tet en sida består av. HTML är den text som bygger upp en sida och som gör att sidan kan ha olika funktioner.

ID

ID är ett attribut man sätter på ett element för att specificera det så att man kan använda sig av just det elementet i t.ex. din CSS. ID kan endast användas en gång per element.

Class

Class är ett attribut likt "ID" som benämning på ett element men skillnaden är att Class kan användas flera gånger inom samma element.

Meta

Meta är en del som endast syns i HTMlen. Meta är en beskrivning på din sida.

Responsive

Responsive är att sidan komprimeras desto mindre den blir. Om man gör sidan mindre ska ändå strukturen på sidan bevaras. Ett exempel är att göra en sida mobilanpassad.

Http

Http är det kommunikationssättet som webbsidor använder sig av för att överföra sina sidor över WWW. Http sätter sig runt informationen som ska skickas och för fram den via World Wide Web.

Https

Https är liknande http men skillnaden är bokstaven "s" som läggs till. Det som bokstaven s gör är att den sätter säkerhet runt datan vilket göt det svårare att läsa datan som skickas över WWW. Det som utgör säkerheten är att den krypteras vilket gör det svårare att läsa av infon.
